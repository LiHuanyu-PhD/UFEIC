# Underwater Fluid Emission Image Captioning Dataset

This repository provides a domain-specific dataset for **underwater fluid emission image captioning** in marine geoscience.

The dataset is designed for underwater vision--language research, marine geological scene understanding, and semantic interpretation of underwater optical observation imagery.

## Overview

The dataset contains underwater fluid emission scenes from cold seeps, hydrothermal systems, hydrocarbon seepage areas, subsea engineering leakage scenes, and non-seepage observation backgrounds.

Each image is annotated with five English captions describing scene entities, fluid-emission processes, environmental background, associated ecology, and observation or facility-related information.
